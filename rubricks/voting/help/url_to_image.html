URL картинок для графика
В этом поле задается url (или относительный путь) к каталогу, в котором находятся картинки для построения графика результатов голосований.
Вы можете использовать свои картинки. Нужно подготовить 10 гифов с именами 0.gif - 9.gif и поместить их в каталог, который задан в поле "URL картинок для графика".
Значение по умолчанию - "http://img.tigvote.ru"
