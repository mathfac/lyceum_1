Язык системных сообщений
Этот параметр указывает, на каком языке будут отображаться системные сообщения (ваш голос был принят, не выбрано ни одного пункта и т.п.). Сами системные сообщения на разных языках хранятся в файле languages.txt, в который Вы можете добавить свой вариант сообщений или написать их на языке, которого еще нет в этом файле. Количество доступных языков будет постепенно пополняться, последнюю версию файла с языками можно скачать здесь: http://www.tigvote.ru/download/.
Если по какой-то причине скрипт не может подгрузить файл languages.txt, то системные сообщения будут отображаться на английском языке.
Значение по умолчанию - "Русский (Windows)"
