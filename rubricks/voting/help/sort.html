Сортировка результатов
Выберите один из вариантов сортировки результатов голосования:
  • не сортировать - пункты голосования отображаются в том порядке, в каком они следуют при создании голосования;
  • кол-во голосов по убыванию - пункты голосования сортируются по убыванию результатов, т.е 10, 7, 4, 3 (сверху вниз);
  • кол-во голосов по возрастанию - пункты голосования сортируются по возрастанию результатов, т.е 3, 4, 7, 10 (сверху вниз);
  • пункты в алфавитном порядке - пункты голосования сортируются в алфавитном порядке, т.е ааа, ббб, ддд, ррр (сверху вниз);
  • пункты в порядке, обратном алфавитному - пункты голосования сортируются в порядке обратном алфавитному, т.е ррр, ддд, ббб, ааа (сверху вниз);
    Значение по умолчанию - "кол-во голосов по убыванию"
