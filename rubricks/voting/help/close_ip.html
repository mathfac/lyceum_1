Запретить повторное голосование с одного IP адреса?
Для того, чтобы пользователь мог голосовать неограниченное количество раз, установите значение этого параметра в "нет". При установленном значении "да" возможны два варианта запрета повторных голосований: если в поле "задержка, после которой можно голосовать повторно" стоит "0", то один IP-адрес может проголосовать только один раз. Если же в поле "задержка, после которой можно голосовать повторно" установить число, отличное от нуля, то ранее голосовавший IP-адрес сможет голосовать повторно по истечении указанного в поле количества минут.
Значение по умолчанию - "Да", задержка - "0"
